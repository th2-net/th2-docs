# Copy Code Button

This plugin adds the `copy-code-btn` Vue component (`CopyCodeBtn.vue`) to all code snippets.

## Transform

Before:

```html
<code class="remark-highlight">
  <pre>...</pre>
</code>
```

After:

```html
<code class="remark-highlight">
  <pre>...</pre>
  <copy-code-btn />
</code>
```
